# SQL for Business Analytics

This project demonstrates the use of SQL for business analytics using SQLite and a housing dataset. The project covers fundamental and advanced SQL concepts including data retrieval, aggregation, grouping, joins, and query optimization.

## Key Features
- Data Import and Exploration
- COUNT, AVG, SUM Aggregations
- GROUP BY Analysis
- INNER JOIN Operations
- Query Optimization using INDEX
- Business Insight Generation

## Tools Used
- SQLite
- DB Browser for SQLite
- GitHub

## Project Outcome
Analyzed housing data to extract meaningful insights and demonstrated SQL techniques commonly used in business analytics and data analysis workflows.
